# Student_Expense_Tracker
A minimalist Python/flask expense tracker for students, focused on the "Needs vs. Wants" methodology for simpler financial oversight and database demonstration.
📊 Minimalist Student Expense Tracker
A streamlined web application designed to help students manage daily spending on essentials like canteen meals and stationery without the complexity of traditional accounting software.

💡 The "Needs vs. Wants" Philosophy
Unlike traditional trackers with dozens of categories, this project simplifies financial health into two primary buckets:

Needs: Non-negotiable spending (Stationery, Course Materials, Basic Meals).

Wants: Discretionary spending (Canteen snacks, Entertainment).

By focusing on this binary classification, the app provides an immediate "Reality Check" on non-essential spending habits.

🛠️ Technical Implementation
Database Connectivity: Built using Django Models and SQLite to demonstrate robust CRUD operations.

Efficient Schema: Optimized with a lightweight 4-column architecture for high performance and low error rates during live demonstrations.

Data Aggregation: Features a custom fetch logic to calculate and display the total sum of "Wants," helping users visualize their saving potential.

🚀 Key Features
Quick Entry: Rapidly insert daily expenses with a single-click category toggle.

Smart Insights: Automatically calculates the percentage of budget spent on "Wants."

Persistent Storage: Full database integration ensuring data is saved across sessions.
